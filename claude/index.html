PRINCIPE CARDINAL

Les trois guerres se renforcent mutuellement. Le récit (舆论战) fournit le cadre et amplifie les victoires juridiques. La dé-sidération (心理战) forme les citoyens qui alimentent les deux autres. Le prétoire (法律战) produit les victoires concrètes qui nourrissent le récit et la mobilisation. La même architecture que Pékin. Inversée éthiquement.
