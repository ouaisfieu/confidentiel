Dossier : Fake News "Deborsu"

Investigation Forensique & Fraude Fiscale

Résumé de l'Incident

Une campagne de désinformation coordonnée cible le journaliste belge Christophe Deborsu. Des articles fabriqués lui attribuent de fausses citations qualifiant les citoyens de "fraudeurs fiscaux". L'objectif est d'inciter au clic vers des plateformes d'investissement frauduleuses. Cette investigation révèle que l'infrastructure repose massivement sur GitHub Pages et l'injection de données structurées JSON-LD pour manipuler le référencement (SEO).

Le Vecteur Technique : JSON-LD Cloaking

Les attaquants utilisent le JSON-LD (JavaScript Object Notation for Linked Data) pour injecter de fausses métadonnées invisibles à l'utilisateur humain mais lues par Google. Cela permet d'afficher "Article de Presse" dans les résultats de recherche alors que la page est une simple redirection vers une arnaque.

Code Source (Simulation) index.html 
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "NewsArticle",
  "headline": "Christophe Deborsu : 'Ils trichent tous !'",
  "image": "fake_img.jpg",
  "author": {
    "@type": "Organization",
    "name": "Le Soir (Faux)"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Media Belge",
    "logo": {
      "@type": "ImageObject",
      "url": "logo.png"
    }
  },
  "datePublished": "2023-10-27"
}
</script>

Recommandations & Contre-Mesures

1

Vérification de l'URL

Toujours vérifier le domaine. Si l'URL contient github.io ou une suite aléatoire de caractères pour un site d'actualité, c'est une arnaque.

2

Signaler à GitHub

Utiliser le formulaire "Report Abuse" de GitHub pour les pages hébergeant du contenu phishing.

3

Analyse JSON-LD

Utiliser l'outil de test des résultats enrichis de Google pour voir si le contenu affiché correspond aux métadonnées.
